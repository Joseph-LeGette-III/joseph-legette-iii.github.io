Transloc Alternate Interface

Problem Statement

The TransLoc app's routes page shows many confusing paths, making it difficult for users to find clear bus routes and reducing use of the service.

Sketches

The sketches show different ways a user could interact with the app. This user searches for a route from Park Place to Pedigru and gets a route and an ETA.

Affinity Diagram

The affinity diagram shows functionality that would be implemented in the app along with barriers and solutions to implementation.
